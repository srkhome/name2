# 勝仁的 LIFF 互動名片

這是一個給「總管家｜襄理｜社宅包租代管顧問 柯勝仁」使用的
手機版互動名片頁面，可放在 GitHub Pages，並透過 LINE LIFF 開啟。

特色：

- 為手機直式瀏覽優化（100% 視窗高）
- 上方頁籤 + 左右滑動四個分頁
- 放入本人照片（main-profile.jpg）
- 支援 LINE 一鍵加入、電話撥打、Email、表單連結
- 適合印在名片上的 QR Code 入口

---

## 📁 專案結構

```bash
liff_card/
├─ index.html          # 主頁面（LIFF 名片）
├─ main-profile.jpg    # 勝仁本人照片（總管家制服）
└─ （可自行新增案例圖片等）
